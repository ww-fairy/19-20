vscode 快捷键：
移动
移动行首home/end
移动代码首 ctrl + home/end
移动单词长度  ctrl + 左右
删除
删除一行光标前Ctrl + Backspace
删除一行光标后 Ctrl + delete
删除一整行     Ctrl + Shift + K
编辑
跳至下行 Ctrl + Enter 
代码向上移动Alt + ↑
代码向上复制 Alt + Shift + ↑
跳转
当前项目工程，全局搜索文件 Ctrl + P
跳转指定行    Ctrl + G
当前文件方法跳转 Ctrl + shift + O
多光标
点击位置插入光标  Alt
每行末尾创建光标  Alt + Shift + I
工作区快捷键
创建多个编辑器  Ctrl + \
在已经打开的文件之间进行切换 Ctrl + Pagedown/Pageup	
显示/隐藏控制台  Ctrl + `
代码
foo（）函数 放上去 「Shift + F12」 看调用
F2 命名重构
小灯泡 方法重构
